# Smart Home AI Assistant

An intelligent home automation platform that uses machine learning to learn user preferences and optimize energy consumption.

## Features
- **Voice Control** — Interact with your smart home using natural language commands
- **Energy Optimization** — ML-based learning reduces energy bills by up to 30%
- **Security Dashboard** — Real-time monitoring with anomaly detection
- **Multi-device Support** — Works with 200+ smart home devices

## Architecture
- **Backend:** Python FastAPI with async workers
- **ML Engine:** TensorFlow + custom energy prediction models
- **Frontend:** React with real-time WebSocket updates
- **Database:** PostgreSQL + Redis caching
